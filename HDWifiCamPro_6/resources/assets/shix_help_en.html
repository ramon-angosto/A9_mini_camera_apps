Help document

 

        1How to add devices to home page

 

        1.1Click the "+" in the upper right corner for the first network connection of the device

 

        1.2The default password of the device is 6666

 

        1.3The speed of the device connecting to the network is related to the network environment

 

 

 

        2How to switch network for devices

 

        2.1If there are other WiFi networks around the device and it is necessary to switch the device to other WiFi networks, switch directly in the WiFi settings

 

        2.2In case of network problems or the need to change the WiFi environment, it is necessary to reconfigure the device. First, long press the "reset" button on the back of the fuselage, repeat the steps of network distribution, and configure a new WiFi network for the device
